forward function is composed by multiple forward streaming modules



.. rubric:: Parameters:

.. list-table::
    :widths: 20 80

    *
        - p_sel

        - determins the port id to read or write

    *
        - p_domain

        - the domain partition object

    *
        - p_sm

        - an array of streaming module objects

    *
        - p_t

        - the start time step in the current process

    *
        - p_src

        - the source wavefiled

    *
        - p_v2dt2

        - the velocity model v^2 * dt^2

    *
        - p_pi0

        - the first input memory of pressure wavefield at t-1

    *
        - p_pi1

        - the second input memory of pressure wavefield at t-1

    *
        - p_po0

        - the first output memory of pressure wavefield at t-1

    *
        - p_po1

        - the second output memory of pressure wavefield at t-1

    *
        - p_ppi0

        - the first input memory of pressure wavefield at t-2

    *
        - p_ppi1

        - the second input memory of pressure wavefield at t-2

    *
        - p_ppo0

        - the first output memory of pressure wavefield at t-2

    *
        - p_ppo1

        - the second output memory of pressure wavefield at t-2

    *
        - p_upb

        - the memory to store the upper-boundary data
