This is full Z\-Standard decompression streaming kernel function. It supports all block sizes and supports window size upto 128\-K\-B. It takes entire Z\-Std compressed file as input and produces decompressed file at the kernel output stream. This kernel does not use D\-D\-R memory, it uses streams instead. Intermediate data is stored in internal B\-R\-A\-Ms and stream F\-I\-F\-Os, which helps to attain better decompression throughput.
