template class xf::security::bls¶

#include "bls.hpp"

Overview¶

BLS curve pairing. This class template provide basic operators in BLS processing.

Parameters:

W Bit length of curve parameters. Parameters should all be smaller than 2^W. 
template <int W>
class bls

// fields

ap_uint <W> a
ap_uint <W> b
ap_uint <W> p
