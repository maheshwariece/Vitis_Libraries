Release Note
============

.. toctree::
   :hidden:
   :maxdepth: 1

1.0
----

The 1.0 release introduces HLS primitives, prebuild kernles and software APIs for RTM (Reverse Time Migration) FDTD (Finite Difference Time Domain) solution. It also includes the activation primitives and fully connected network kernels for MLP.
