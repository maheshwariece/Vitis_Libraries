	    unsigned int deviceIndex
	    )

.. rubric:: Parameters:

.. list-table::
    :widths: 20 80

    *
        - A

        - pointer to matrix A in the host memory

        - kernelIndex

        - index of kernel that is being used, default is 0

    *
        - xfblasStatus_t

        - 1 if the library was not initialized

    *
